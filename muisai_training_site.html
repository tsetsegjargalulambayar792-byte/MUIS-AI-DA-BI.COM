🎯 Цогц сургалт - 4 өдөр

AI үндсэн ойлголт → Excel advanced + Python өгөгдөл дүрслэл ба шинжилгээ → BI ойлголт → Tableau → SPSS Modeler машин сургалт → AI стратеги → Generative AI → Ёс зүй → Ирээдүйн чиг хандлага зэрэг бүх агуулгыг шат дараатай, уялдаа холбоотой судалж, бүх програмыг практик жишээн дээр бодит өгөгдлөөр ажиллана.

📊 Модуль 1: AI + Excel Advanced + BI + Tableau

AI болон Excel-ийн advanced аргачлалуудыг BI ойлголттой хослуулж, Power BI, Excel Power Pivot, Power View ашиглан BI тайлан гаргах, Tableau дээр интерактив дашбоард бүтээх.

🤖 Модуль 2: AI + Excel Advanced + Машин сургалт + SPSS Modeler

AI үндсэн ойлголт, Excel өгөгдөл бэлтгэх, SPSS Modeler ашиглан машин сургалтын загвар бүтээж, таамаглал хийх (Decision Tree, Random Forest, Logistic Regression гэх мэт).

📈 Модуль 3: AI + BI + Tableau + Машин сургалт + SPSS Modeler

AI үндсэн ойлголт, BI тайлан, дашбоард хийх, Tableau Analytic, SPSS Modeler програмд модел боловсруулах, таамаглах.

🐍 Модуль 4: Excel Advanced + Python + AI Tools

Excel-ийн Power Query, Pivot Table, Dynamic Arrays, What-If Analysis, Solver, Macro, VBA зэрэг advanced техникүүдийг ашиглан өгөгдөл бэлтгэж, Python (Pandas, Matplotlib, Seaborn) ашиглан шинжилгээ хийх, Generative AI тайлан, контент үүсгэх.

🚀 Модуль 5: AI + Машин сургалт + SPSS Modeler + Python

AI үндсэн ойлголт, Generative AI - Prompt Engineering хэрэгжүүлэх, Python өгөгдөл бэлтгэл, машин сургатын загвар боловсруулах.
